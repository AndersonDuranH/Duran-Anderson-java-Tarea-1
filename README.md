## DESCRIPCION:
Este proyecto consiste en una aplicación de consola desarrollada en Java que reúne múltiples ejercicios y funcionalidades básicas de programación en un solo menú interactivo. El objetivo principal del proyecto es poner en práctica conceptos fundamentales de la programación orientada a objetos, estructuras de control, manejo de datos y uso de arreglos y matrices.

La aplicación presenta un menú con diferentes opciones que permiten al usuario seleccionar la operación o ejercicio que desea ejecutar. Cada módulo trabaja de manera independiente y aborda un tema específico de programación.

Las funcionalidades incluidas en el sistema son:

Calculadora básica
Permite realizar operaciones matemáticas simples como suma, resta, multiplicación y división utilizando datos ingresados por el usuario.
Número par o impar
Evalúa un número introducido por el usuario para determinar si es par o impar mediante el uso de operadores condicionales.
Tabla de multiplicar
Genera la tabla de multiplicar de un número seleccionado, utilizando estructuras repetitivas.
Contador de vocales
Analiza una palabra o frase ingresada y cuenta la cantidad de vocales presentes.
Promedio de notas
Calcula el promedio de varias calificaciones y determina el resultado final del estudiante.
Clase Estudiante
Implementa una clase orientada a objetos para almacenar y mostrar información de estudiantes utilizando atributos y métodos.
Cuenta bancaria
Simula operaciones básicas de una cuenta bancaria, como almacenamiento de datos del titular y manejo de información financiera.
Herencia de vehículos
Demuestra el uso de herencia en Java mediante clases relacionadas con vehículos y sus características.
Matriz 3x3
Permite trabajar con matrices bidimensionales realizando ingreso y visualización de datos.
Sistema básico de inventario
Gestiona productos de forma sencilla mediante almacenamiento y visualización de información de inventario.
Salir de la aplicación
Finaliza la ejecución del programa.
